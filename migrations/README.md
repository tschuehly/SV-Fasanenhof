# Migrations

Dieses Verzeichnis sammelt Rohmaterial aus den alten Websites und aus der Migrationsphase.

## Struktur

- `raw-content/sv-fasanenhof/`: manuell aufbereitete Rohinhalte der alten Hauptvereinsseite
- `raw-content/bsg-fasanenhof/`: manuell aufbereitete Rohinhalte der alten BSG-Seite
- `crawl-output/`: Crawl-Snapshots der alten Seiten
- `content-archive/`: archivierte Altseiten und Hilfstexte
- `content-maps/`: frühere Planungs- und Mapping-Dateien

## Wichtige Regel

Nichts aus diesem Verzeichnis wird direkt vom SSG gerendert. Die produktive Site liest ausschließlich aus `content/`.
